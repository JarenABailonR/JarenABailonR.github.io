Lab 003 · Hardening Debian 13

Checklist básico para asegurar un servidor recién instalado: usuarios, SSH, firewall y auto-updates.

1. Actualizar sistema

sudo apt update && sudo apt full-upgrade -y

2. Usuario sudo

sudo adduser anghelo
sudo usermod -aG sudo anghelo

3. Configurar SSH

En /etc/ssh/sshd_config:

PermitRootLogin no
PasswordAuthentication yes # o 'no' si usas keys
X11Forwarding no
sudo systemctl restart ssh

4. Firewall UFW

sudo apt install -y ufw
sudo ufw allow OpenSSH
# sudo ufw allow 80/tcp
sudo ufw enable

5. Servicios inútiles

sudo ss -tulpn # Ver puertos abiertos
sudo systemctl disable --now <servicio>

6. Updates automáticas

sudo apt install -y unattended-upgrades
sudo dpkg-reconfigure unattended-upgrades
