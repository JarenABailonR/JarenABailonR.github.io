Lab 001 · Fix DNS en Debian 13

Objetivo: resolver el error Fallo temporal al resolver deb.debian.org configurando correctamente los DNS.

1. Síntoma

El comando apt update falla al intentar resolver el repositorio:

sudo apt update
# ...
# Fallo temporal al resolver <deb.debian.org>

2. Causa raíz

El archivo /etc/resolv.conf no existía o no tenía servidores DNS configurados, impidiendo la resolución de nombres de dominio.

3. Remediación aplicada

Crear un /etc/resolv.conf mínimo con DNS públicos:

sudo sh -c 'cat > /etc/resolv.conf << "EOF"
nameserver 1.1.1.1
nameserver 8.8.8.8
EOF'

4. Verificación

ping -c 4 debian.org
sudo apt update

Nota: En producción podrías usar un resolver interno en lugar de DNS públicos.
